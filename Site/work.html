Affine

This program will perform the functions of an affine cipher on the files you provide it. These are the functions it can do:
  • Encrypt will change the file you provide or create one based off your given A and B values into an encrpyted version.
  • Derypt will change the file you provide or create one based off your give A and B values into a decrypted version.
  • Decipher will take in your file and a dictionary file then it will attempt to determine the A and B values used to encode it.

    • Here is the link to its Github.

Asteroids

This program will run a game similar to the old arcade game of the same name.
  • W, A, S and, D on your keyoard will move your spaceship. The spacebar will fire bullets.
  • You may have to press and release the controls in order for them to be recieved.

    • Here is the link to its Github.
